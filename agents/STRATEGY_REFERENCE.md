## 0. 커리큘럼이 관통하는 대원칙 (전 부서 공통)

1. **재료·수급·시장 방향이 차트/지표보다 항상 우선**한다. 지표 1개 = 가능성, 3개 이상 동일 신호 = 확률 높은 타점. (Phase 4·5·7)
2. **패턴 발견 즉시 진입 금지 → 확인 캔들 대기**. 거래량 없는 신호는 신뢰도 낮음. (Phase 4)
3. **나갈 기준(익절가·손절가) 없으면 진입하지 않는다.** 청산 기준은 진입 전 확정하고 바꾸지 않는다. (Phase 8·9)
4. **확실한 것만 진입한다.** 지표·수급·차트 충돌 시 = 비중 축소 또는 관망. (Phase 5·7)
5. **세력 의심 / C급 재료 / SNS·후발주 = 진입 금지.** 예외 없음. (Phase 6)

## 1. 정보부 — 스크리닝 (`agents/intel/screening`)

대응 커리큘럼: Phase 6(재료/테마), Phase 7-7(스크리닝), Phase 8 STEP 2~3.

### 기본 필터 (Phase 7-7, 8-5)
- `[△]` 거래량 전일 대비 **300% 이상**(수급 폭발 필터) — 현 스크리닝은 거래대금 랭킹+5요소 점수. 거래량 급증 비율을 명시 가산 요소로 강화 후보.
- `[✗]` 시총 밴드 **500억~5조원**, 주가 **2,000~100,000원** 필터 — 현재 명시적 시총/주가 밴드 필터 없음. 후보.
- `[✓]` 투자경고/투자위험·관리종목·거래정지 제외 — `scorer.dart_penalty` 관리종목류 -100 하드 필터(§19 폴백에서도 부활 금지)와 정합.
- `[✓]` 최종 후보 3개 이하로 압축 → 단일 집중(§5.7) `EntrySelector`가 최강 1종목 선택과 정합.

### 테마/선도주 (Phase 6-4, 6-10)
- `[△]` **선도주만 진입, 후발주 금지**: 테마 거래량 1위 + 직접 사업 연관 + (과거 선도주 이력). 현 `ThemeDetector`는 종목명 키워드 기반 7개 테마. "거래량 1위=선도주" 판정 로직은 미구현 → 후보.
- `[✗]` **테마 사이클 5단계**(태동→폭발→확산→분산→소멸). 진입=2단계 초중반, 청산=4단계 초. 거래량 3일 연속 감소 = 끝물. → 스크리닝/메모리 신호로 후보.
- `[✗]` **재료 강도 4단계(S/A/B/C)**. C급(소문/SNS/커뮤니티) 진입 금지, S~A급만 정상~최대 비중. 현재 재료 등급 분류 없음(백테스트는 DART 미사용 §17.2). → 후보.

### 선반영 / 뉴스 (Phase 6-2, 6-11)
- `[✗]` **Buy the Rumor, Sell the News**: 발표 전 +15~20% 선상승 시 추격 금지. 호재 발표 직후 하락 = 선반영 완료. → 후보.
- `[✗]` 노이즈 필터: 공식 채널(공시/보도) 여부, 오늘 처음 나온 재료 여부, 소스 명확성. SNS/유튜브 종목 = 진입 금지. → 후보.

> 스크리닝 점수 핵심 매핑은 CLAUDE.md §2.2.1. 본 절의 `[✗]`/`[△]`는 5요소 비중을 흔드므로 §13-6(백테스트+승인) 대상.

- `[△]` **RSI 50 기준선 / 60 돌파 모멘텀**: 50 위=상승 우세(홀딩 근거), 60 돌파+거래량=가속. 현재 30~70 전환만 사용 → 50/60 활용 추가 후보.
- `[✗]` **RSI·MACD·OBV 다이버전스**: 신고가-지표고점 낮아짐=음적(익절), 신저가-지표저점 높아짐=양적(반전). 진입/청산 정밀도 향상 후보.
- `[✗]` **볼린저밴드**: 수축(폭발 예고, 수축 중 진입 금지)·밴드워킹(추세=홀딩, 역방향 금지)·평균회귀(횡보 한정). `core/indicators/bollinger.py` 존재 → 신호 결합 후보.

- `[✗]` **OBV**: 방향 포함 누적 거래량. 가격↑+OBV↓=취약(익절), 가격↓+OBV↑=매집(반전). `core/indicators`에 OBV 추가 후보.
- `[✗]` **스토캐스틱**: RSI와 중복이므로 **둘 다 쓰지 않는다**(커리큘럼 명시). 도입 비권장.

## 4. 리스크부 (`agents/risk/risk_manager`)

대응 커리큘럼: Phase 8 STEP 5~7, Phase 9 손절, Phase 11(심리).

### 게이트 (§2.4, §4)
- `[✓]` 동시보유 ≤3(HL-01), 연속손절 3회→1시간 쿨다운(HL-02), 진입 시간대 화이트리스트(HL-03/04), 슬리피지 5틱(HL-05), 신용 담보비율(HL-06).

### R/R (Phase 8-8) — 강한 보강 후보
- `[✗]` **R/R 비율 게이트**: (목표가-진입가)/(진입가-손절가). 최소 1:1.5, 권장 1:2. R/R 1.0 미만 = NO-GO. **수수료 0.015%×2 + 거래세 0.18%(매도)** 포함 실질 R/R로 계산. 현 리스크부는 R/R 게이트 부재 → 진입 게이트에 추가 강력 후보(하드리밋 아님, `strategy_params` 튜너블로 도입 가능).
- 참고: 진입 시 분석부가 `entry_candle_low`(손절 기준)를 이미 실어 보냄(§2.3) → 손절폭 산출 가능, 목표가(TP1)와 결합하면 R/R 계산 가능.

### 사이징 (Phase 6-1 재료 강도별 비중, Phase 10-1)
- `[✓]` 고정 상한 없는 동적 사이징(신호강도·시장등급·ATR·신용). (§4.1, §2.4)
- `[△]` **재료 강도별 비중**: S급 최대~C급 진입금지. 재료 등급 입력 생기면(§1) 사이징 보조 후보.

### 심리 게이트 (Phase 8-2) — 룰 치환
